# 機能006: 進捗統計表示

## 概要
完了済み・未完了・合計タスク数を表示して進捗を可視化する機能。

## ユーザーストーリー
「子供として、今日どれくらい宿題が終わったかを数字で確認して達成感を得たい」

## 受け入れ基準
- [ ] 「完了: X個」「残り: X個」「合計: X個」が表示される
- [ ] タスクの完了・追加・削除時に統計が自動更新される
- [ ] フィルター適用時は表示中のタスクの統計が表示される
- [ ] 視覚的に分かりやすいデザイン

## 技術詳細
- タスク配列からの集計計算
- リアルタイム更新機能
- フィルター状態に応じた統計計算

## 表示項目
- 完了済みタスク数
- 未完了タスク数  
- 合計タスク数
- 完了率（オプション）

## 統計計算ロジック
```javascript
completed = tasks.filter(task => task.completed).length
remaining = tasks.filter(task => !task.completed).length
total = tasks.length
```

## 実装優先度
**低** - 補助的な機能

## 依存関係
- 機能001: ローカルデータストレージ
- 機能002: タスク一覧表示
- 機能003: タスク完了切り替え

## 完了条件
- 統計数値が正確に表示される
- タスク状態変更時に自動更新される
- フィルター適用時の統計も正確
- 見やすいデザインで配置されている
