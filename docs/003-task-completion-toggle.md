# 機能003: タスク完了切り替え

## 概要
タスクをクリック・タップして完了・未完了を切り替える機能。

## ユーザーストーリー
「子供として、宿題が終わったときに簡単にタップして完了にマークしたい。間違えたときは元に戻したい」

## 受け入れ基準
- [ ] タスクのチェックボックスをタップして完了・未完了を切り替えられる
- [ ] 完了状態の変更が即座に視覚的に反映される
- [ ] 変更がLocalStorageに自動保存される
- [ ] タッチデバイスで操作しやすいサイズのクリック領域

## 技術詳細
- クリックイベントハンドラー
- タスクオブジェクトのcompleted状態の更新
- DOM要素のクラス追加・削除
- LocalStorageの更新

## UI動作
1. チェックボックスをタップ
2. 即座に見た目が変更（✓マーク、線消し等）
3. バックグラウンドでデータ保存

## 実装優先度
**高** - 主要なユーザーアクション

## 依存関係
- 機能001: ローカルデータストレージ
- 機能002: タスク一覧表示

## 完了条件
- タップ操作で完了状態が切り替わる
- 変更がLocalStorageに保存される
- 視覚的フィードバックが適切
- iPad/iPhoneで操作しやすい
