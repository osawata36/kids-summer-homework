# 機能004: タスク追加

## 概要
新しい宿題タスクを追加する機能。テキスト入力とカテゴリ選択を含む。

## ユーザーストーリー
「子供として、新しい宿題が出たときに簡単にアプリに追加したい。何の教科かも選べるようにしたい」

## 受け入れ基準
- [ ] テキスト入力フィールドでタスク内容を入力できる
- [ ] カテゴリ選択ボタンでカテゴリを選択できる
- [ ] 「追加」ボタンでタスクが一覧に追加される
- [ ] 追加後、入力フィールドがクリアされる
- [ ] 空のタスクは追加できない（バリデーション）
- [ ] 追加されたタスクがLocalStorageに保存される

## 技術詳細
- フォーム要素（input, button）
- フォームサブミット処理
- バリデーション機能
- 一意IDの生成（UUIDまたはタイムスタンプ）

## UI要素
- タスク内容入力フィールド
- カテゴリ選択ボタン群
- 追加ボタン
- バリデーションメッセージ

## バリデーションルール
- タスク内容: 1文字以上必須
- カテゴリ: 選択必須
- 重複チェック: なし（同じタスクでも追加可能）

## 実装優先度
**高** - 基本的な入力機能

## 依存関係
- 機能001: ローカルデータストレージ
- 機能002: タスク一覧表示

## 完了条件
- テキスト入力とカテゴリ選択が機能する
- バリデーションが適切に動作する
- 追加されたタスクが即座に一覧に表示される
- データがLocalStorageに保存される
