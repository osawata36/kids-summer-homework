# 機能005: カテゴリフィルター

## 概要
カテゴリ別にタスクをフィルタリングして表示する機能。

## ユーザーストーリー
「子供として、算数の宿題だけ、国語の宿題だけを見たいときに、簡単に絞り込みたい」

## 受け入れ基準
- [ ] 「すべて」「算数」「国語」「理科」「その他」のフィルターボタンがある
- [ ] フィルターボタンをタップすると該当カテゴリのタスクのみ表示される
- [ ] 「すべて」を選択すると全カテゴリのタスクが表示される
- [ ] 現在選択中のフィルターが視覚的に分かる
- [ ] フィルター状態はページリロード後も保持される

## 技術詳細
- フィルター状態の管理
- DOM要素の表示・非表示制御
- LocalStorageでのフィルター状態保存
- アクティブボタンのスタイリング

## UI動作
1. フィルターボタンをタップ
2. 該当カテゴリのタスクのみ表示
3. 他のタスクは非表示（display: none）
4. ボタンの見た目がアクティブ状態に変更

## カテゴリ一覧
- すべて（デフォルト）
- 算数
- 国語  
- 理科
- その他

## 実装優先度
**中** - 利便性向上機能

## 依存関係

- 機能002: タスク一覧表示

## 完了条件

- フィルター状態の視覚的表示が明確

- ページリロード後もフィルター状態が復元される
