# 機能002: タスク一覧表示

## 概要
保存されたタスクを一覧形式で表示し、完了・未完了の状態を視覚的に分かりやすく表示する機能。

## ユーザーストーリー
「子供として、今ある宿題の一覧を見て、どれが終わっていてどれが残っているかを一目で分かるようにしたい」

## 受け入れ基準
- [ ] タスクが追加順または作成日時順で表示される
- [ ] 完了済みタスクには視覚的な違い（チェックマーク、線消し等）がある
- [ ] 未完了タスクは通常表示される
- [ ] カテゴリ名がタスクと一緒に表示される
- [ ] タスクが0個の場合は適切なメッセージが表示される

## 技術詳細
- HTMLリスト要素での表示
- CSSクラスによる完了・未完了の区別
- 動的DOM操作でのタスク描画

## UI要素
- タスクテキスト
- カテゴリラベル
- 完了状態アイコン
- チェックボックス（クリック可能）

## 実装優先度
**高** - 基本的な表示機能

## 依存関係
- 機能001: ローカルデータストレージ

## 完了条件
- LocalStorageからタスクを読み込んで表示される
- 完了・未完了の視覚的区別が明確
- レスポンシブデザインでiPad/iPhoneに最適化されている
